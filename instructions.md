こんにちは。これからあなたと協力して、PythonのStreamlitとOpenAI APIを使った高度なWebアプリケーションを開発します。このプロジェクトは、専門家の持つ「暗黙知」を「形式知」に変換することを目的としています。

**プロジェクト名:** Process Insight Modeler (PIM)

**最終的なゴール:**
ユーザーが記述した「生産プロセス」の概要をインプットとし、アプリケーションが対話を通じてプロセスを「機能」と「実体（ノード）」に分解。最終的に、ノード間の相互作用を定量化した有向行列（ネットワーク）を生成し、プロセスのボトルネックや改善点を可視化・分析する。このプロジェクトの核心は、専門家の頭の中にある暗黙的なプロセス知識を、テキスト、リスト、行列、ネットワークといった異なるデータ構造間でシームレスに変換し、それぞれをユーザーが直感的に操作・理解できる形で視覚化することです。

**技術スタック:**
- **UI:** Streamlit
- **言語:** Python
- **LLM:** OpenAI API (gpt-4oなどを想定)
- **データ操作:** Pandas, NumPy
- **ネットワーク分析/可視化:** NetworkX, Matplotlib, Seaborn

**開発アプローチ:**
これから私が提示する機能要件に基づき、ステップバイステップで開発を進めます。各ステップは独立して動作確認ができるようにします。あなたは私の優秀なペアプログラマーとして、要件を深く理解し、クリーンで拡張性の高いコードを生成してください。特に、Streamlitのセッションステート(`st.session_state`)を効果的に活用した状態管理が重要になります。
